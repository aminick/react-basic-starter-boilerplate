# React Basic Starter Boilerplate
A basic webpack starter boilerplate for React projects.
## Get started
### Installation
```
git clone https://github.com/aminick/react-basic-starter-boilerplate.git
npm install
```
### Usage
**Development**
```
npm start
```
**Production**
```
npm run build
```
## Features
* Webpack
* Babel
* Sass
* PostCSS
